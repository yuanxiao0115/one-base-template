### 结论

- `ObTable` 已完成 `tableKey` registry 语义收口：默认 key 改为组件实例 `uid` 派生值，`props.tableKey` 可覆盖，`getTableRef()` 按 key 获取实例，避免同页多表默认 key 冲突。
- `adaptive` 已按 pure 预期切到“视口贴底优先 + 容器高度兜底”，并补齐 `ResizeObserver + window.resize` 监听与清理。
- `getTableDoms()` / `setHeaderSticky()` 已增强容错：优先 `$el` 根节点 + 多 selector fallback + fixed header 批量 sticky。
- 分页已区分 controlled 与 default 语义：受控 `currentPage/pageSize` 与 `defaultCurrentPage/defaultPageSize` 不再混用；`total > 0` 时不再透传 `pageCount`。
- 文档与源码门禁测试已同步更新。
