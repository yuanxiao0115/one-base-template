# P2 路线（上手）

## 适合对象

- 首次进入仓库，对目录和规则不熟。
- 目标是“先做成一件事”，不是先理解所有设计细节。

## 30-45 分钟目标

1. 本地跑起 `admin-lite`。
2. 能创建一个新后台 app。
3. 能跑通最小验证命令。

## 最小必读（按顺序）

1. [/guide/quick-start](/guide/quick-start)
2. [/guide/admin-lite-base-app](/guide/admin-lite-base-app)
3. [/guide/menu-route-spec](/guide/menu-route-spec)（先看开头 TL;DR 速查卡）
4. [/guide/for-users](/guide/for-users)

## 一次完整练习（直接照做）

```bash
pnpm install
pnpm dev:admin-lite
pnpm new:app my-admin --preset standard
pnpm -C apps/my-admin typecheck
pnpm -C apps/my-admin lint
pnpm -C apps/my-admin build
```

## 做功能时只记这三条

1. 模块契约：`index.ts（moduleMeta） + routes.ts`。
2. 列表页基线：`ObPageContainer + ObTableBox + ObTable`。
3. 提交前至少跑：`typecheck + lint + build`。

## 升级到 P4 的信号

- 你可以不看他人示例，独立完成一个简单管理模块（list/search/create/edit）。
- 你知道在哪个页面找“路由规范、模块规范、验证门禁”。
